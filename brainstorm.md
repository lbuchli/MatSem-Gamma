# Brainstorm and Project Ideas

Die Gamma-Funktion Γ(z) erweitert die Fakultät n! auf beliebige Argumente. Es gilt Γ(n) = (n-1)!. Die Funktion ist als komplexe Funktion durch wenige, natürliche Eigenschaften eindeutig festgelegt.

## Research

- The gamma function is an extension of the factorial function.
- Γ(n) = (n-1)! → The gamma function is offset by one.
- Γ(-0.5) = sqrt(π)

- Information Entropy can be expressed as $H(X) = \alpha + \ln \theta + \ln \Gamma (\alpha )+(1-\alpha )\psi (\alpha )$

- The volume of an $n$-Dimensional ball is $ \frac{\pi^(n/2)}{\Gamma(n/2)} $
  - Sphere packing is used e.g. in error correction (but I think no direct connection?)

- The gamma distribution
  [occurs](https://en.wikipedia.org/wiki/Gamma_distribution#Occurrence_and_applications),
  among other things, when modelling:
  - Size of rainfalls, cancer incidence, repair prediction for hydraulic systems, ...
